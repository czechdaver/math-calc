# Installation

## Prerequisites
- Node.js 18+
- npm or yarn

## Setup

1. Clone the repository:
   ```bash
git clone https://github.com/yourusername/math-calc.git
cd math-calc
   ```

2. Install dependencies:
   ```bash
npm install
# or
yarn
   ```

3. Start the development server:
   ```bash
npm run dev
# or
yarn dev
   ```

4. Open http://localhost:3000 in your browser.
